# Securedesk DAM

## Cómo arrancar el proyecto

1. Instalar XAMPP
2. Crear la carpeta securedesk-dam en C:\xampp\htdocs
3. Crear la estructura: /public, /app, /views, /db, /docs
4. en /public, crear el archivo /index.php de prueba de arranque
5. Iniciar Apache en XAMPP
6. Verificar acceso a http://localhost/securedesk-dam/public
